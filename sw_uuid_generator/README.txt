This directory contains the Interop Demo for Smallworld 5.x

For legal reasons, the source code of the Interop demonstration showing 
how to interface Magik to the JFreechart library has been withdrawn.

The example code for interfacing to the JFugue libary is included.

Please refer to the user documentation on Interop for information on 
how to write an interoperability module to interface to a 3rd-party library.
